== README

This shopping cart is a practice project at DaVinci Coders.

You can clone it and play with it by running the following commands:

 $ git clone git@github.com:mkreyman/shopping_cart.git
 $ cd davinci_shop
 $ bundle exec rake db:create:all db:test:clone
 $ rails s

 Then navigate to http://localhost:3000 with your browser.
